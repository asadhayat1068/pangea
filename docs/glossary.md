# Pangea Glossary (ubiquitous language)

- **Tenant** — a business using Pangea (e.g. FreshCart). All data is scoped to a tenant.
- **Merchant** — a user acting on behalf of a tenant who manages catalog/orders.
- **Product** — an item a tenant offers for sale. Always belongs to exactly one tenant.
- **Catalog** — the collection of a tenant's products.
- **Customer** — an end user who will browse and buy (not in Sprint 1).

Rule: PM, BA, and engineers use these exact words to mean exactly these things.
